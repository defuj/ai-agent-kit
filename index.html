CLI

One CLI to Rule Them All

Cross-platform commands to manage your entire agent ecosystem.

Initialize a project

Install 31 agents into any OpenCode project.

terminal 
$ npx opencode-agent-kit init
# Run inside your project directory
# macOS / Linux / Windows (WSL)
  ✓ Copying .opencode/ configuration
  ✓ Merging opencode.json
  ✓ Installing dependencies
  ✓ Setting up agentmemory
  ✦ 31 agents ready. Run 'opencode' to start.

# Windows (PowerShell)
PS> npx opencode-agent-kit init

Upgrade agents & skills

Keep your agent ecosystem up to date.

terminal 
$ npx opencode-agent-kit upgrade
# All platforms (macOS / Linux / Windows)
  ✓ Checking npm registry...
  ✓ Current version: v1.2.0
  ✓ Upgraded to latest
  ✦ opencode-agent-kit is up to date

Diagnose your setup

Check system health and fix common issues.

terminal 
$ npx opencode-agent-kit doctor
# macOS / Linux / Windows
  ✓ Node.js v18+
  ✓ opencode.json found
  ✓ .opencode/ directory complete
  ✓ agentmemory installed

# Windows (PowerShell)
PS> npx opencode-agent-kit doctor

Uninstall cleanly

Remove agents, skills, and config — no leftover cruft.

terminal 
$ npx opencode-agent-kit uninstall
# All platforms (macOS / Linux / Windows)
  ! This will remove .opencode/ and opencode.json
  ! Type 'y' to confirm: y
  ✓ Removing .opencode/ directory
  ✓ Removing opencode.json
  ✓ Cleaning .gitignore
  ✦ opencode-agent-kit removed from project

# Windows (PowerShell)
PS> npx opencode-agent-kit uninstall
